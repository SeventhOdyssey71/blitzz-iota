### Project Summary
BLITZZ is a comprehensive DeFi platform built on IOTA that combines an automated market maker (AMM) DEX, meme token launchpad with bonding curves, and advanced trading features. The platform leverages IOTA's feeless transactions and Move smart contracts to provide users with efficient swaps, liquidity provision, and token creation capabilities, all while maintaining a seamless user experience with accurate blockchain-based calculations and real-time pool discovery.

### Project Description
BLITZZ represents a significant advancement in the IOTA DeFi ecosystem by providing a full-stack decentralized finance solution that addresses key market needs:

**Core Components:**
1. **AMM DEX**: Our decentralized exchange implements a constant product formula with a 0.3% swap fee, enabling efficient token swaps with accurate price calculations based on real-time pool reserves. The system automatically discovers and tracks liquidity pools, ensuring users always get the best rates.

2. **Meme Token Launchpad**: A revolutionary token creation platform with bonding curve mechanics, allowing anyone to launch tokens with just 2 IOTA. Tokens automatically graduate to the main DEX after reaching 4,000 IOTA in liquidity, creating a fair launch environment.

3. **Advanced Trading Features**: Beyond basic swaps, we're implementing limit orders and dollar-cost averaging (DCA) strategies, bringing CEX-like functionality to DeFi.

**Why IOTA:**
- **Feeless Transactions**: IOTA's unique architecture eliminates gas fees, making micro-transactions and frequent trading viable
- **Move Smart Contracts**: Provides security and composability for complex DeFi operations
- **Scalability**: IOTA's DAG structure ensures our platform can handle high-frequency trading without congestion
- **Growing Ecosystem**: Contributing to IOTA's expanding DeFi landscape with innovative solutions

**Technical Innovation:**
- Real-time pool discovery and tracking system
- Accurate swap calculations using actual blockchain data
- Seamless wallet integration with IOTA dApp Kit
- Modern UI/UX with React 19 and Next.js 15
- Comprehensive testing suite for smart contracts
